# Data Analysis Portfolio

Este repositório contém projetos de análise de dados desenvolvidos
com Python, Pandas e Jupyter Notebook.

## Projetos

### 1. Análise Exploratória Básica
- Limpeza de dados
- Estatísticas descritivas
- Insights iniciais

### 2. Comparação entre Grupos
- Análise por cidade
- Comparações de salário e idade

### 3. Visualização de Dados
- Gráficos para apoio à análise (em desenvolvimento)

## Ferramentas
- Python
- Pandas
- Jupyter Notebook
